Global Country Indicators Dashboard

## About This Project

This Power BI dashboard was created to explore and visualize key economic and demographic indicators from countries around the world. The goal was to turn raw data into meaningful insights that make it easier to understand global trends, compare countries, and identify patterns in areas such as population, GDP, and economic growth.

Whether you're a student, analyst, researcher, or simply interested in global data, this dashboard provides an interactive way to explore and compare country-level statistics.

---

## Purpose

The purpose of this project is to:

* Understand global economic and population trends.
* Compare countries using key performance indicators.
* Explore regional differences and development patterns.
* Practice data modelling, visualization, and dashboard design in Power BI.
* Present complex data in a simple and interactive format.

---

## Tech Stack

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Data Modelling
* Interactive Visualizations

---

## Data Source

The dashboard uses country-level indicator data that includes:

* Population
* GDP
* GDP Per Capita
* Population Density
* Economic Growth Metrics
* Regional Information

The dataset contains information from multiple countries and regions, allowing for meaningful global comparisons and analysis.

---

## Key Features

* Interactive filters and slicers for easy exploration.
* Country-by-country performance comparison.
* Regional analysis and breakdowns.
* KPI cards highlighting important metrics.
* Trend analysis to identify growth patterns.
* Map visualizations for geographic insights.
* Drill-down capabilities for deeper analysis.

---

## Business Value

This dashboard demonstrates how data can be transformed into actionable insights.

Some of the questions it helps answer include:

* Which countries have the highest GDP and GDP per capita?
* How does population impact economic performance?
* Which regions are experiencing the strongest growth?
* How do countries compare across different indicators?

By presenting data visually, users can quickly identify trends and make informed decisions without having to analyze large spreadsheets manually.
